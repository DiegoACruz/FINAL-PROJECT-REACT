# Paradise Nursery

Paradise Nursery is a simple e-commerce web application where users can browse different categories of plants and add them to a shopping cart.

## Project Description

The Paradise Nursery application allows users to explore a collection of indoor plants, succulents, and flowering plants. Users can view product information, add plants to a shopping cart, modify quantities, and see the total cost of their selected items.

This project was built using React to demonstrate the creation of a modern front-end web application with reusable components and state management.

## Technologies Used

- React
- React Router
- JavaScript
- HTML
- CSS

## Installation

To run this project locally:

1. Clone the repository

```
git clone https://github.com/DiegoACruz/FINAL-PROJECT-REACT.git
```

2. Go to the project folder

```
cd FINAL-PROJECT-REACT
```

3. Install dependencies

```
npm install
```

4. Start the project

```
npm start
```

The application will run at:

```
http://localhost:3000
```

## Features

- Display plants grouped by category
- Add plants to the shopping cart
- Disable the "Add to Cart" button after adding a product
- Show the number of items in the cart icon
- Increase or decrease item quantity
- Remove items from the cart
- Display the total price of the cart
